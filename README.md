# AlleGrowth — Landing Page Allegro Ads

Statyczna strona ofertowa dla usługi prowadzenia kampanii **Allegro Ads** z trzema pakietami.

## Pakiety

| Pakiet | Cena | Zakres |
|--------|------|--------|
| **Start** | 199 zł / mies. | Prowadzenie i optymalizacja kampanii Allegro Ads |
| **Wzrost** | 399 zł / mies. | Start + raporty + wskazówki co poprawić (tytuły, grafiki) |
| **Dominacja** | 599 zł / mies. | Wzrost + propozycje tytułów, analiza konwersji aukcji, wskazywanie błędów na ofertach |

> Wszystkie ceny netto. Budżet reklamowy Allegro nie jest wliczony w cenę pakietu.

## Technologia

Czysty **HTML + CSS + JS** w jednym pliku `index.html`. Bez frameworków i bez procesu budowania — gotowe do hostowania w dowolnym miejscu.

## Uruchomienie lokalne

Otwórz `index.html` w przeglądarce. To wszystko.

## Publikacja (GitHub Pages)

1. Wypchnij repozytorium na GitHub (np. przez GitHub Desktop).
2. W repo wejdź w **Settings → Pages**.
3. W sekcji *Build and deployment* wybierz **Branch: `main` / root** i zapisz.
4. Po chwili strona będzie dostępna pod adresem `https://<twoja-nazwa>.github.io/AlleGrowth/`.
